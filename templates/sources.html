Sources & Project Credits

Citations for the tools, languages, and standards used while building this site.

Programming Frameworks & Tools

  • Python (v3.10+): Backend programming logic. Documented at docs.python.org.
  • Flask (v3.0.0): Python web framework used for routing and rendering templates. Documented at flask.palletsprojects.com.
  • Jinja2: Templating engine used to build pages from a shared base layout. Documented at jinja.palletsprojects.com.
  • SQLite3: Built-in Python library for database management. Used for persistent storage of items, claims, and users. See sqlite.org.
  • Werkzeug: WSGI utility library, used specifically for secure password hashing (generate_password_hash) and safe file-name handling on uploads (secure_filename). See werkzeug.palletsprojects.com.

Languages & References

  • HTML, CSS & JavaScript: Front-end structure, styling, and interactivity. Reference: MDN Web Docs (developer.mozilla.org).
  • Google Fonts — Bitter: Typeface used for the site logo/wordmark. See fonts.google.com.

Design & Accessibility Standards

  • W3C Web Content Accessibility Guidelines (WCAG 2.1 AA): Site built with semantic HTML (<nav>, <main>, <header>, <footer>), ARIA labels, a skip-to-content link, keyboard-operable menus and dialogs, and AA-level color contrast. See w3.org/WAI.
  • WAI-ARIA Authoring Practices: Used to guide the accessible dialog (modal) and navigation patterns. See w3.org/WAI/ARIA/apg.
  • Responsive CSS: Layout managed entirely through CSS Flexbox, CSS Grid, and media queries — no external UI libraries such as Bootstrap.

Project & Topic

  • FBLA Website Coding & Development: 2025–2026 competitive event topic, "School Lost-and-Found Website," created in partnership with code.org.
