Sources & Project Credits

Citations for tools, languages, and standards used during development.

Programming Frameworks & Tools

  • Python (v3.10+): Backend programming logic. Documented at docs.python.org
  • Flask (v3.0.0): Python web framework used for routing and rendering templates. Documented at flask.palletsprojects.com
  • SQLite3: Built-in Python library for database management. Used for persistent data storage of items and users.
  • Werkzeug: Web Server Gateway Interface (WSGI) web application library, used specifically for secure password hashing and secure file uploads.

Design & Accessibility Standards

  • W3C Web Content Accessibility Guidelines (WCAG): Site developed using ARIA labels, semantic HTML (<nav>, <main>, <header>), and high color contrast. See w3.org/WAI
  • Responsive CSS: Layout managed entirely through CSS Flexbox, CSS Grid, and Media Queries without relying on external UI libraries like Bootstrap.
